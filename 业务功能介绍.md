## 一、产品定位

**项目投资管理系统** 是一款面向**项目型、投资型、贸易型企业**的一体化经营管理平台。

它以"**项目**"为核心，把企业经营中最关键的六条业务线——
**项目、投资、销售、采购、财务、报表**——全部整合进同一个系统，
实现"**一处录入、处处可用、实时归集、自动核算**"。

老板看的是利润，财务管的是资金，业务跑的是单据，客户要的是透明。
本系统让这四方诉求在**同一个数据底座**上得到满足。

## 三、功能模块详解

### 1. 经营看板（Dashboard）— 一屏掌控全局
- **四大核心指标**：项目总数、收入合同金额、支出合同金额、综合毛利率
- **可视化图表**：项目状态环形图，直观掌握进行中 / 已完成 / 已暂停分布
- **最近操作流**：团队动态实时滚动，谁在做什么一目了然
- **报表快捷入口**：经营驾驶舱、经营预警、费用盈亏、销售回款、合同执行一键直达

### 2. 项目管理 — 以项目为核算单元
- 项目档案与全生命周期状态管理
- 单项目下汇总合同、采购、销售、费用、发票等全部业务数据
- **项目盈亏测算**：收入 − 成本 − 费用，毛利与毛利率自动计算

### 3. 投资管理 — 资金与分红清晰透明
- 投资人 / 参与人档案与出资记录
- 分红规则配置、分红批次执行、分红明细报表
- 投资回报全程可追溯

### 4. 销售管理 — 从签约到回款全打通
- **销售合同**：合同金额、条款、执行进度集中管理
- **销售出库单**：出库明细与合同、客户关联
- **销售回款**：回款登记与对账，应收一目了然

### 5. 采购管理 — 采购入库与对账一体化
- 采购单录入与采购合同管理
- 运输 / 采购明细 Excel 批量导入
- **对账中心**：采购入库与供应商账目自动核对，减少错漏

### 6. 费用管理 & OCR 智能录入 — AI 提效
- 项目费用归集与多级费用分类树
- **OCR 票据识别**：上传发票/票据图片，自动提取金额、税号、抬头等字段
- Excel 批量导入，支持大批量历史数据迁移

### 7. 合同 & 发票管理 — 单据合规可控
- 收入合同 / 支出合同集中归档，支持 Word 合同导入
- 发票列表管理与发票 OCR 导入
- 发票与合同勾稽，防止重复报销与漏开

### 8. 报表分析中心 — 数据驱动决策
- **经营驾驶舱**：全局经营指标总览
- **经营预警**：异常指标主动提醒
- **费用盈亏 / 销售回款 / 合同执行漏斗 / 客户资金**等多张专业报表
- 支持多维筛选、图表可视化与一键导出

### 9. 客户协同门户 — 与客户共赢
- 独立客户登录入口，与内部系统隔离
- 客户自助查看交付记录、对账单、系统消息
- **充值审核**与**余额扣减同步**，预存款资金往来全透明

### 10. 系统管理 — 安全与合规
- 多角色权限体系（管理员 / 财务 / 经理 / 客户协同专员 / 客户）
- 账号管理、操作日志审计
- 数据库定时自动备份，数据安全无忧

## 四、典型业务流程

```
签订合同 ──▶ 采购入库 ──▶ 销售出库 ──▶ 费用归集 ──▶ 回款/分红 ──▶ 报表分析
   │            │            │            │            │            │
   └─ 合同管理   └─ 采购+对账  └─ 销售订单  └─ OCR 录入   └─ 投资分红   └─ 经营驾驶舱
                                                                       （实时毛利）
```

每个环节产生的数据都自动汇入对应项目，最终在**经营看板**与**报表中心**实时呈现，
管理者无需等待财务出表，随时掌握**真实经营状况**。
