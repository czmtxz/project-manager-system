## ✨ 为什么选择它？

> **告别 Excel 满天飞、对账靠记忆、利润算不清的时代。**

传统项目型企业最头疼的，是**钱花在哪、利润有多少、客户欠多少**这三笔糊涂账。
本系统把分散在表格、微信、纸质票据里的经营数据，全部收口到**一个平台**，
从**合同签订 → 采购入库 → 销售出库 → 费用报销 → 回款分红 → 报表分析**，
形成完整、可追溯、实时更新的业务闭环。

| 痛点 😫 | 我们的解法 ✅ |
| --- | --- |
| 项目成本算不清 | 项目维度归集收入/支出/费用，**毛利实时可视** |
| 发票合同手工录入慢 | **OCR 智能识别**，拍照/上传即自动填单 |
| 销售采购两头乱 | 销售出库、采购入库、对账中心**一体化联动** |
| 客户资金不透明 | **客户协同门户**，充值、扣减、对账客户自助查 |
| 老板要数据等半天 | **经营驾驶舱**一屏看全局，预警自动提醒 |

## 🧩 核心功能模块

### 📊 经营看板（Dashboard）
- 项目总数、收入/支出合同金额、综合毛利率**四大核心指标**实时刷新
- 项目状态环形图、最近操作动态一目了然
- 报表中心快捷入口：经营驾驶舱、经营预警、费用盈亏、销售回款……

### 📁 项目管理
- 项目全生命周期管理（进行中 / 已完成 / 已暂停）
- 单项目维度归集合同、采购、销售、费用、发票，**项目盈亏一键测算**

### 💰 投资管理
- 投资人 / 参与人档案管理
- 出资记录、分红规则、分红批次与分红报表

### 📈 销售管理
- 销售合同、销售回款、销售出库单全流程
- 出库单与对账联动，**回款进度清晰可控**

### 🛒 采购管理
- 采购单录入、采购合同、运输明细导入
- **对账中心**：采购入库与供应商账目自动核对

### 🧾 费用管理 & OCR 智能录入
- 项目费用归集、费用分类树
- **发票 / 合同 OCR 识别**：上传图片自动提取金额、税号、抬头等关键字段
- 支持 Excel 批量导入，告别手工录入

### 📑 合同 & 发票
- 收入/支出合同集中管理，支持 Word 合同导入
- 发票列表、发票 OCR 导入、发票与合同勾稽

### 📉 报表分析中心
- 经营驾驶舱、经营预警、费用盈亏、销售回款、合同执行漏斗、客户资金……
- 多维筛选 + 图表可视化 + 一键导出

### 🤝 客户协同门户
- 独立客户登录入口，客户可自助查看交付、对账、消息
- 充值审核、余额扣减同步，**资金往来全透明**

### 🛡️ 系统管理
- 多角色权限（管理员 / 财务 / 经理 / 客户协同专员）
- 账号管理、操作日志审计、数据库自动备份

## 🛠️ 技术栈

| 层 | 技术 |

## ⚡ 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

> Windows 用户也可直接双击 `一键启动项目管理系统.bat`。

### 3. 访问系统

打开浏览器访问 👉 `http://127.0.0.1:5000`

| 角色 | 默认账号 | 默认密码 |
| --- | --- | --- |
| 管理员 | `admin` | `admin123` |

> ⚠️ **首次登录后请立即修改默认密码！**
> 客户协同专员使用同一登录页，登录后自动进入客户协同管理。

## 📂 项目结构

```
项目管理系统/
├── app.py                    # 主应用入口（路由与核心逻辑）
├── route_extensions.py       # 扩展路由
├── reports_routes.py         # 报表路由
├── report_service.py         # 报表数据服务
├── ocr_utils.py              # OCR 识别工具
├── client_collab_ops.py      # 客户协同业务
├── auth_utils.py             # 认证与权限
├── templates/                # Jinja2 页面模板
│   ├── login.html            # 登录页（宣传海报背景）
│   ├── dashboard.html        # 经营看板
│   └── ...                   # 各业务模块页面
├── static/
│   ├── css/style.css         # 全局样式
│   └── img/login-bg.png      # 登录页宣传海报
├── migrations/               # 数据库迁移脚本
├── backups/                  # 自动备份目录
├── uploads/                  # 上传文件目录
└── requirements.txt          # Python 依赖
```
